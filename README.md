## My process

I started by building the html structure. I used the figma files to plan out what elements I would need. I tried to think about the CSS as I was building the HTML file. I then created the CSS file and added the CSS reset code. I decided to use a custom reset instead of copying one of the common ones.  This helps me understand what all my code is doing and why. Next I looked at the HTML file with only the CSS reset and started styling.

### What I learned

In this project I was continuing to increase my CSS and html knowledge. I try to think about how I am structuring my html and how I am organizing CSS code. This is not so important on a smaller project like this, but I don't have a clear organizational plan for larger projects and things often get out of hand. Thinking about this now will help me to form a system as my projects get larger.
